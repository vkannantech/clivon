# üëã Welcome

**Clivon YouTube Desktop** is not just another wrapper. It is a re-engineered consumption engine built on **Tauri v2** and **Rust**. 

We rejected the industry standard of bloated Electron apps (which eat 500MB+ RAM for a simple tab) and built a **Native Window Architecture** that gives you the raw performance of a system browser with the utility of a dedicated application.
